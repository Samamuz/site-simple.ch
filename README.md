# site-simple.ch


## Présentation

**site-simple.ch** est un site web dont l’objectif est de proposer une présence en ligne claire, rapide et efficace. Il met l’accent sur la simplicité, l’accessibilité et la facilité de maintenance.

## Objectifs

* Proposer une interface simple et intuitive
* Offrir un chargement rapide et une navigation fluide
* Être facilement administrable et évolutif
* Garantir une compatibilité mobile (responsive design)

## Public cible

* Indépendants et petites entreprises
* Associations ou projets personnels
* Toute personne souhaitant un site web simple et fonctionnel

## Fonctionnalités principales

* Pages statiques (Accueil, À propos, Contact, etc.)
* Design responsive
* Optimisation SEO de base
* Formulaire de contact
* Hébergement et nom de domaine (.ch)

## Technologies utilisées

*(À adapter selon le projet réel)*

* HTML5 / CSS3
* JavaScript
* Framework CSS léger (ex. Tailwind, Bootstrap) ou CSS personnalisé
* Hébergement web standard
